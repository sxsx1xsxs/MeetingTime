Summer


Thanks to the first class, we teamed up as 'ColdCode', and began to discuss and sketch our web application.


• We decided to develop a scheduling web application for organizing meetings.

• We came up with main features to implement by brainstorming user stories.

• Our wire-frames then came out.




Fall


The 1st iteration of our project. We worked really hard day and night. 4AM library will always be in our memory of this project.


• We implemented the basic logic flow and framework.

• Login module was achieved by using Google APi.

• Creating events and selecting time slots worked out well.




Winter


Time flies and we went on fighting for the 2nd Iteration. Although temperature keeps going down everyday, our passion has never decreased.


• New feature: organizers can make decisions.

• Notification center & email invitation are now achieved

• UI was greatly improved and that is not the ending point :).
